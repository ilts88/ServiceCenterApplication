# ServiceCenterApplication
Это веб-приложение является системой сервисного центра компании электроники. Система представляет собой решение для взаимодействия клиентов, которым требуется ремонт, с сотрудниками сервисного центра. С помощью этого приложения можно получать запросы на ремонт от клиентов, обрабатывать их, а также сообщать клиентам о процессе выполнения ремонта. 

Приложение разделено на две части. Первая часть доступна всем пользователям. На ней они могут создавать запросы на ремонт и просматривать статусы запросов. Вторая часть доступна только администраторам сайта. Для того, чтобы получить к ней доступ, требуется перейти по адресу /adminpage (в интерфейсе системы специально нет явного упоминания об этой странице) и ввести логин и пароль. В этом разделе приложения доступно управление поступившими запросами на ремонт.

В дополнение к функционалу обработки запросов в системе есть раздел публикации новостей компании. Он также разделён на две части. В первой части, доступной всем пользователям, можно просматривать опубликованные новости. Во второй части, доступной только администраторам, можно управлять публикацией новостей.

Веб-приложение создано на основе фреймворка ASP.NET Core 8.0 и настроено на подключение к базе данных MySQL. Для работы с базой данных используется Entity Framework Core, подключение реализуется с помощью провайдера Pomelo.EntityFrameworkCore.MySql. Для настройки подключения к базе данных используется файл конфигурации DatabaseConfiguration.json. Также можно настроить данные для входа администратора с помощью файла конфигурации AdminAccessConfiguration.json.
